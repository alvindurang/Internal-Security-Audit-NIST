# Internal Security Audit: Botium Toys (NIST CSF)

## Objective
To perform an internal security audit for a fictional company to identify risks and ensure compliance with NIST CSF standards.

## Tools & Frameworks
*   **Framework**: NIST Cybersecurity Framework (CSF)
*   **Concepts**: CIA Triad, Risk Mitigation, Compliance (GDPR/PCI-DSS)

## Methodology
1. **Identify**: Listed critical assets and current security controls.
2. **Protect**: Recommended improvements for access control and data encryption.
3. **Detect**: Outlined the need for centralized logging.

## Findings & Recommendations
- **Gap**: Lack of MFA on administrative accounts.
- **Fix**: Implementation of 2FA across all employee logins.
